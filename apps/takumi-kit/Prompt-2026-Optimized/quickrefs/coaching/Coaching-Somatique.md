# QuickRef: Coaching Somatique

> Coaching par le corps : ancrage, régulation, transformation incarnée.

---

## Fondements

**Soma** (grec) = Corps vivant, ressenti de l'intérieur

Principes clés :
- Le corps stocke les expériences (mémoire somatique)
- Les schémas corporels influencent les schémas mentaux
- La transformation durable passe par le corps

---

## Les 3 Cerveaux

### Cerveau Reptilien (Survie)
```
- Réponses automatiques : fight/flight/freeze/fawn
- Régulation physiologique de base
- Ressenti : tension, contraction, urgence
```

### Cerveau Limbique (Émotions)
```
- Attachement, mémoire émotionnelle
- Réponses relationnelles
- Ressenti : chaleur, connexion ou fermeture
```

### Néocortex (Pensée)
```
- Raisonnement, planification
- Narration, sens
- Ressenti : clarté ou confusion
```

**Clé** : Réguler le reptilien AVANT d'engager le néocortex

---

## Techniques de Régulation

### Ancrage (Grounding)
```
1. Pieds au sol, sentir le contact
2. Scanner les points d'appui (pieds, siège, dos)
3. Respiration lente (4-4-4-4)
4. Nommer 5 choses visibles
```

### Respiration Cohérence Cardiaque
```
- Inspirer 5 secondes
- Expirer 5 secondes
- 5 minutes minimum
- Effet : équilibre système nerveux
```

### Shake & Tremor
```
- Secouer le corps entier (30 sec - 2 min)
- Laisser les tremblements naturels
- Permet la décharge de stress accumulé
```

### Posture de Puissance (Power Pose)
```
- Posture ouverte, expansive
- 2 minutes minimum
- Effet : augmente confiance, réduit cortisol
```

## Fenêtre de Tolérance

```
      Hyperactivation
      (anxiété, agitation)
   ────────────────────────
   |                      |
   |   ZONE OPTIMALE      |  ← Objectif : élargir
   |   (présence, calme)  |
   |                      |
   ────────────────────────
      Hypoactivation
      (dissociation, freeze)
```

**Signes hyperactivation** : cœur rapide, respiration courte, agitation
**Signes hypoactivation** : engourdissement, déconnexion, fatigue

## Application Outils Numériques

### Check-in Corporel (App)
```
"Où es-tu en ce moment ?"
[ ] Hyperactivé (agité, anxieux)
[ ] Zone optimale (présent, calme)
[ ] Hypoactivé (fatigué, déconnecté)

→ Proposer exercice adapté
```

### Exercices Guidés
```
- Audio ancrage (2-5 min)
- Respiration guidée avec animation
- Body scan progressif
- Mouvement libre avec musique
```

### Tracking Somatique
```
- Journal quotidien : "Où dans ton corps ?"
- Mapping émotions-zones corporelles
- Évolution de la fenêtre de tolérance
```
