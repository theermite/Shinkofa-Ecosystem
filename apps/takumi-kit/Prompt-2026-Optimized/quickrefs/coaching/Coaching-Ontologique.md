# QuickRef: Coaching Ontologique

> Coaching centré sur l'être, les croyances et la transformation identitaire.

---

## Fondements

**Ontologie** = Étude de l'être, de ce qui constitue notre réalité

Le coaching ontologique travaille sur :
- **L'observateur** que nous sommes (pas les problèmes)
- Les **distinctions** qui façonnent notre perception
- La **cohérence** corps-émotion-langage

---

## Les 3 Domaines de l'Être

### 1. Langage (Linguistique)
```
Ce que nous disons ↔ Ce que nous créons

- Actes de langage : affirmer, déclarer, promettre, demander
- Conversations internes (dialogue interne)
- Histoires que nous nous racontons
```

**Questions puissantes** :
- "Quelle histoire te racontes-tu sur cette situation ?"
- "Si tu changeais cette interprétation, qu'est-ce qui serait possible ?"

### 2. Émotions (Émotionnalité)
```
Les émotions comme prédispositions à l'action

- Peur → Fuite/Protection
- Colère → Limite franchie
- Tristesse → Perte/Deuil
- Joie → Expansion/Connexion
```

**Questions puissantes** :
- "Quelle émotion est présente en ce moment ?"
- "Que te dit cette émotion sur ce qui est important pour toi ?"

### 3. Corps (Corporalité)
```
Le corps comme ancrage et mémoire

- Posture physique influence l'état mental
- Tensions = émotions non exprimées
- Corps précède souvent la pensée
```

**Questions puissantes** :
- "Où ressens-tu cela dans ton corps ?"
- "Quelle posture adopterais-tu si tu étais confiant ?"

---

## Processus Ontologique (OSAR)

```
Observateur → Système → Action → Résultat

1. Quel OBSERVATEUR es-tu de cette situation ?
2. Quel SYSTÈME (croyances, émotions) opère ?
3. Quelles ACTIONS ce système génère-t-il ?
4. Quels RÉSULTATS obtiens-tu ?

→ Pour changer les résultats, changer l'observateur
```

---

## Distinctions Clés

| Concept | Distinction |
|---------|-------------|
| **Fait vs Interprétation** | "Il pleut" vs "C'est une mauvaise journée" |
| **Déclaration vs Affirmation** | Créer une réalité vs Décrire une réalité |
| **Demande vs Exigence** | Possibilité de refus vs Obligation |
| **Plainte vs Réclamation** | Sans action vs Avec demande de changement |

## Application Outils Numériques

### Questions de Réflexion (App)
```
- "Quel observateur suis-je aujourd'hui ?"
- "Quelle déclaration veux-tu faire ?"
- "Quelle demande as-tu besoin de formuler ?"
```

### Journaling Guidé
```
1. Situation (faits neutres)
2. Interprétation (ma lecture)
3. Émotion (ce que je ressens)
4. Corps (où je le ressens)
5. Action possible (nouveau choix)
```

### Feedback Non-Violent
```
"Je remarque [fait observable]
et je ressens [émotion]
parce que j'ai besoin de [besoin]
serais-tu d'accord pour [demande] ?"
```
