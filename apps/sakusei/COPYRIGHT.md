## À Propos

**Podcast The Ermite** - Application web d'enrichissement audio avec fréquences thérapeutiques et ambiances méditatives.

Cette application a été développée par **Jay "The Ermite" Goncalves** pour offrir un outil accessible permettant d'enrichir des fichiers audio (podcasts, méditations, formations) avec :
- Fréquences thérapeutiques (432 Hz, 528 Hz, binaurales)
- Ambiances méditatives (8 musiques incluses)
- Export audio haute qualité

**Philosophie** : La Voie Shinkofa (真の歩 - "Le Véritable Pas") - Une approche holistique du développement personnel intégrant Design Humain, neurodiversité et technologies modernes.

---

## Technologies Utilisées

- **Frontend** : React 18 + Vite + Tailwind CSS
- **Audio Processing** : Web Audio API
- **PWA** : Progressive Web App (offline support)
- **Hébergement** : o2Switch (web statique)
