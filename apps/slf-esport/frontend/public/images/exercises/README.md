# Exercise Thumbnails

## Required Images

This directory should contain thumbnail images for all cognitive exercises.

### Format
- Format: PNG ou JPG
- Dimensions: 600x400px (3:2 ratio)
- Taille: < 200KB par image
- Style: Cohérent avec la charte graphique SLF

### Images Nécessaires

#### Mémoire (4)
- `memory-cards.png` - Jeu de memory cards
- `pattern-recall.png` - Rappel de motifs colorés
- `sequence-memory.png` - Mémoire de séquence
- `image-pairs.png` - Paires d'images

#### Réflexes & Attention (3)
- `reaction-time.png` - Test de temps de réaction
- `peripheral-vision.png` - Vision périphérique
- `multitask.png` - Multitasking

#### Gaming MOBA (3)
- `last-hit-trainer.png` - Entraînement last hit
- `dodge-master.png` - Esquive de projectiles
- `skillshot-trainer.png` - Entraînement skillshot

#### Bien-être (1)
- `breathing.png` - Exercice de respiration

## Fallback Actuel

En attendant les vraies images, les exercices utilisent:
- Emojis de grande taille (fonctionnel)
- Gradient de couleur primaire en fond

## Création des Images

Recommandations:
- Utiliser les couleurs SLF (#004225, #FF9800, #FFD600)
- Style gaming/e-sport
- Icônes claires et lisibles
- Éviter le texte dans les images
- Optimiser pour le web (compression)
