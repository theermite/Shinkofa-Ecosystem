# QuickRef: Bushido & Ninjutsu Moderne

> Application moderne des principes du Bushido et du Ninjutsu.

---

## Les 7 Vertus du Bushido

### 1. 義 Gi (Rectitude/Justice)
- Faire ce qui est juste, même si difficile
- Intégrité dans les décisions
- **Application** : Code éthique, pas de raccourcis moraux

### 2. 勇 Yū (Courage)
- Agir malgré la peur
- Courage moral > physique
- **Application** : Oser l'authenticité, feedback honnête

### 3. 仁 Jin (Bienveillance)
- Compassion active
- Aide sans attente de retour
- **Application** : UX inclusive, accessibilité prioritaire

### 4. 礼 Rei (Respect/Étiquette)
- Politesse sincère
- Reconnaissance de la dignité d'autrui
- **Application** : Communication respectueuse, pas de condescendance

### 5. 誠 Makoto (Honnêteté)
- Parole = Engagement
- Pas de promesses vides
- **Application** : Features promises = features livrées

### 6. 名誉 Meiyo (Honneur)
- Responsabilité de ses actions
- Réputation par l'intégrité
- **Application** : Assumer erreurs, transparence totale

### 7. 忠義 Chūgi (Loyauté)
- Fidélité aux engagements
- Constance dans les relations
- **Application** : Fiabilité, continuité du service

## Principes Ninjutsu Modernes

### Adaptabilité (変化 Henka)
- S'adapter au contexte
- Flexibilité sans perdre l'essence
- **Application** : Features modulables, personnalisation

### Persévérance (忍耐 Nintai)
- Patience stratégique
- Endurance face aux obstacles
- **Application** : Itérations progressives, pas de big-bang

### Discrétion (隠密 Onmitsu)
- Efficacité silencieuse
- Impact sans ostentation
- **Application** : Performance sans bloat, fonctionnel > flashy

### Préparation (備え Sonae)
- Anticiper les scénarios
- Toujours un plan B
- **Application** : Gestion d'erreurs, fallbacks, backups

### Observation (観察 Kansatsu)
- Comprendre avant d'agir
- Analyse contextuelle
- **Application** : Audit complet avant développement

## Application au Développement

### Code comme Kata (形)
```
- Pratique délibérée
- Répétition pour maîtrise
- Chaque projet = opportunité d'amélioration
```

### Le Vide (空 Kū)
```
- Esprit ouvert, sans préjugés
- Chaque problème = nouveau regard
- Pas d'attachement aux solutions passées
```

### Zanshin (残心) - Vigilance Continue
```
- Attention après l'action
- Tests après déploiement
- Monitoring continu
```
