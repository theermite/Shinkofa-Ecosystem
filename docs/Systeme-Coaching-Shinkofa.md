# Système de Coaching Shinkofa

> Une approche holistique qui honore l'être dans sa totalité : esprit, cognition et corps.

---

## Fondements Philosophiques

### Les Quatre Sources Fondatrices

Le coaching Shinkofa puise dans quatre traditions de sagesse :

**1. Sankofa (Tradition Akan, Ghana)**
- *"Se wo were fi na wosankofa a yenkyi"* - Regarder en arrière pour mieux avancer
- Honorer les ancêtres et les leçons du passé
- Réconciliation avec l'histoire personnelle
- Transmission consciente aux générations futures

**2. Bushido & Ninjutsu (Japon)**

| Vertu | Kanji | Application Coaching |
|-------|-------|---------------------|
| Gi (義) | Rectitude | Alignement valeurs-actions |
| Yu (勇) | Courage | Traverser les peurs |
| Jin (仁) | Compassion | Auto-bienveillance |
| Rei (礼) | Respect | Rituels structurants |
| Makoto (誠) | Sincérité | Authenticité radicale |
| Meiyo (名誉) | Honneur | Intégrité personnelle |
| Chugi (忠義) | Loyauté | Engagement envers soi |

*Ninjutsu Moderne* : Adaptabilité, discrétion stratégique, conscience 360°

**3. Neuroplasticité (Science Moderne)**
- Le cerveau se remodèle à tout âge
- Micro-habitudes créent macro-changements
- Répétition intentionnelle forge nouveaux circuits
- Bases : Théorie polyvagale, Somatic Experiencing, neurosciences contemplatives

**4. Sagesses Spirituelles Ancestrales (Monde)**
- Kabbale : Structure de l'être, arbre de vie
- Philosophies orientales : Tao, équilibre yin-yang, qi
- Traditions chamaniques : Guérison holistique, vision élargie
- Philosophie Jedi : Maîtrise de soi, service, équilibre, sagesse

### Inspirations Complémentaires Intégrées

## Architecture Tri-Dimensionnelle Intégrée

Le coaching Shinkofa n'est PAS trois approches séparées, mais une méthode unifiée qui tisse constamment les trois dimensions ensemble selon le contexte et le besoin.

```
                    ┌─────────────────┐
                    │   ONTOLOGIQUE   │
                    │   (L'Être)      │
                    └────────┬────────┘
                             │
              ┌──────────────┼──────────────┐
              │              │              │
              ▼              │              ▼
    ┌─────────────────┐      │     ┌─────────────────┐
    │  TRANSCOGNITIF  │◄═════╪════►│    SOMATIQUE    │
    │  (La Pensée)    │      │     │   (Le Corps)    │
    └────────┬────────┘      │     └────────┬────────┘
             │               │              │
             └───────────────┼──────────────┘
                             │
                    ┌────────▼────────┐
                    │   COACHING      │
                    │   SHINKOFA      │
                    │   UNIFIÉ        │
                    └─────────────────┘
```

### Dimension 1 : Ontologique

*Exploration de l'être, du sens et de l'identité*

**Objectifs**
- Clarifier la mission de vie
- Aligner valeurs et actions quotidiennes
- Réconcilier avec le passé (Sankofa)
- Définir une vision authentique

**Techniques**
| Technique | Description | Durée |
|-----------|-------------|-------|
| Questionnement socratique | Questions ouvertes révélatrices | 15-30 min |
| Ligne de vie | Cartographie événements clés | 60 min |
| Clarification valeurs | Hiérarchisation des 10 valeurs core | 45 min |
| Vision board guidé | Visualisation future authentique | 30 min |
| Rituel Sankofa | Lettre aux ancêtres/passé | Variable |

**Indicateurs de Progression**
- Clarté mission de vie (échelle 1-10)
- Alignement perçu valeurs-actions
- Paix avec le passé

---

### Dimension 2 : Transcognitif

*Optimisation des fonctions exécutives et patterns mentaux*

**Objectifs**
- Améliorer attention et concentration
- Renforcer mémoire de travail
- Développer flexibilité cognitive
- Installer routines productives

**Spécialisation Neurodiversité**

| Profil | Focus Principal | Stratégies Clés |
|--------|-----------------|-----------------|
| **TDAH** | Attention, impulsivité | Timeboxing, externe rewards, body doubling |
| **TSA** | Flexibilité, transitions | Prévisibilité, scripts sociaux, zones de confort |
| **HPI** | Ennui, surpensée | Challenges complexes, meta-cognition |
| **Hypersensible** | Surcharge, régulation | Filtrage stimuli, pauses stratégiques |

**Techniques**
| Technique | Application | Fréquence |
|-----------|-------------|-----------|
| Pomodoro adapté | Cycles travail/pause personnalisés | Quotidien |
| Brain dump | Décharge mentale structurée | 2x/jour |
| Chunking | Décomposition tâches complexes | Par projet |
| Implementation intentions | "Si X alors Y" | Hebdomadaire |
| Review métacognitive | Analyse patterns pensée | Hebdomadaire |

**Indicateurs de Progression**
- Temps de concentration soutenue
- Tâches complétées vs initiées
- Réduction procrastination perçue

---

### Dimension 3 : Somatique

*Intelligence corporelle et régulation du système nerveux*

**Objectifs**
- Développer conscience intéroceptive
- Réguler système nerveux autonome
- Libérer tensions et traumas stockés
- Ancrer les apprentissages dans le corps

**Fondements Scientifiques**
- Théorie polyvagale (Stephen Porges)
- Mémoire somatique (Peter Levine)
- Embodied cognition (recherche actuelle)

**Techniques**
| Technique | Description | Durée |
|-----------|-------------|-------|
| Scan corporel | Exploration sensations | 10 min |
| Respiration 4-7-8 | Activation parasympathique | 3 min |
| Grounding 5-4-3-2-1 | Ancrage sensoriel | 5 min |
| Shaking/TRE | Libération tension | 10-15 min |
| Mouvement intuitif | Expression corporelle libre | Variable |

**États du Système Nerveux**

| État | Signes | Intervention |
|------|--------|--------------|
| **Ventral vagal** (sécurité) | Calme, connecté, ouvert | Maintenir, explorer |
| **Sympathique** (mobilisation) | Anxieux, agité, défensif | Régulation, respiration |
| **Dorsal vagal** (immobilisation) | Figé, dissocié, épuisé | Activation douce, mouvement |

**Indicateurs de Progression**
- Variabilité fréquence cardiaque (HRV)
- Temps de retour au calme post-stress
- Conscience intéroceptive (échelle MAIA)

## Protocole de Session

### Structure Standard (60 min)

```
1. CHECK-IN (10 min)
   - État actuel (corps, émotion, mental)
   - Cycles énergétiques du jour
   - Intentions session

2. EXPLORATION (35 min)
   - Dimension principale du jour
   - Techniques adaptées au profil
   - Intégration inter-dimensionnelle

3. ANCRAGE (10 min)
   - Rituel de clôture somatique
   - Engagement micro-action
   - Préparation intersession

4. DEBRIEF (5 min)
   - Insights clés
   - Ajustements plan
```

### Adaptation selon Énergie

| Niveau Énergie | Approche | Dimensions |
|----------------|----------|------------|
| **Haute** | Exploration profonde | Ontologique + Transcognitif |
| **Moyenne** | Équilibrée | Toutes proportionnées |
| **Basse** | Soutien doux | Somatique prioritaire |
| **Critique** | Régulation pure | Somatique uniquement |

## Les 7 Sphères d'Intervention

Le coaching Shinkofa accompagne l'individu sur toutes les dimensions de sa vie de manière holistique :

| Sphère | Description | Exemples d'Accompagnement |
|--------|-------------|---------------------------|
| **1. Spirituelle/Quantique** | Connexion à la transcendance, sens de la vie | Mission de vie, pratiques contemplatives, synchronicités |
| **2. Mentale** | Optimisation cognitive et psychologique | Croyances limitantes, patterns de pensée, clarté mentale |
| **3. Émotionnelle** | Intelligence et régulation émotionnelles | Gestion du stress, expression émotionnelle, résilience |
| **4. Physique** | Santé et vitalité corporelle | Énergie, sommeil, mouvement, alimentation consciente |
| **5. Relationnelle** | Harmonisation des liens sociaux | Communication, limites saines, connexion authentique |
| **6. Professionnelle** | Épanouissement dans le travail | Alignement carrière, productivité, créativité |
| **7. Environnementale** | Équilibre avec son écosystème | Espace de vie, nature, impact écologique |

### Approche Trauma-Informed (Obligatoire)

Chaque session intègre les principes du coaching informé par le trauma :

**Window of Tolerance (Dan Siegel)**
- Identifier la zone de régulation optimale
- Techniques de retour à la fenêtre
- Élargissement progressif de la capacité

**Théorie Polyvagale (Stephen Porges)**
- État de neuroception sécurité
- Activation parasympathique ventrale
- Co-régulation avec le coach

**Somatic Experiencing (Peter Levine)**
- Décharge de l'énergie traumatique
- Titration des sensations corporelles
- Ressourcement et résilience naturelle

**Trauma Intergénérationnel (lien Sankofa)**
- Héritage ancestral conscientisé
- Patterns familiaux inconscients
- Guérison de la lignée par l'individu

## Intégration du Profil Holistique Complet

Le coaching Shinkofa s'adapte à l'ensemble du profil holistique de l'utilisateur, pas uniquement au Design Humain.

### Les 5 Dimensions du Profil Holistique

**1. Design Humain**

| Type | Approche Session | Timing Optimal |
|------|------------------|----------------|
| **Générateur** | Réponse à propositions | Quand énergie sacrale disponible |
| **Générateur Manifestant** | Propositions claires avec espace réaction | Informer puis répondre |
| **Projecteur** | Reconnaissance explicite | Sur invitation, sessions courtes |
| **Manifesteur** | Espace pour informer | Selon leur rythme, respect autonomie |
| **Réflecteur** | Cycles lunaires respectés | Décisions fin de cycle, perspectives multiples |

**2. Astrologie Occidentale**
- Thème natal : Forces, défis, potentiels
- Transits actuels : Périodes favorables/challengeantes
- Maisons : Domaines de vie à explorer
- Application : Timing des sessions, focus thématique

**3. Astrologie Orientale (Chinoise)**
- Signe annuel : Tempérament fondamental
- Élément : Approche énergétique (Bois, Feu, Terre, Métal, Eau)
- Compatibilités : Dynamiques relationnelles
- Application : Compréhension des cycles de vie

**4. Numérologie**
- Chemin de vie : Mission fondamentale
- Nombre d'expression : Comment on se manifeste
- Année personnelle : Thème de l'année en cours
- Application : Orientation des objectifs, compréhension des cycles

**5. Personnalité et Neurodivergence**

| Profil | Focus Principal | Adaptations Coaching |
|--------|-----------------|---------------------|
| **TDAH** | Attention, impulsivité, fonctions exécutives | Timeboxing, external rewards, body doubling, micro-tâches |
| **TSA** | Flexibilité, transitions, particularités sensorielles | Prévisibilité, scripts, zones de confort, routines claires |
| **HPI** | Ennui, surpensée, intensité | Challenges complexes, méta-cognition, profondeur |
| **Hypersensible** | Surcharge, régulation émotionnelle | Filtrage stimuli, pauses stratégiques, validation |
| **Multipotentiel** | Navigation passions multiples | Intégration, cycles, permission d'explorer |

### Synthèse du Profil Holistique

Le coach Shinkofa (humain ou Shizen) établit une synthèse unique pour chaque individu :

```
PROFIL HOLISTIQUE =
  Design Humain (type + stratégie + autorité)
  + Astrologie Occidentale (soleil + lune + ascendant + transits)
  + Astrologie Orientale (signe + élément)
  + Numérologie (chemin de vie + année personnelle)
  + Profil Neurodivergent (forces + besoins + stratégies)
```

Cette synthèse guide :
- Le style de communication
- Le rythme des sessions
- Les techniques privilégiées
- Le timing des interventions
- Les objectifs réalistes
